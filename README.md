# Инструкция по начальной настройке конфигурации «Управление IT-фирмой для компании «Ваш компьютерный мастер»

Данная инструкция предназначена для администраторов системы и настройки использования новой функциональности.


------
 
## 1. Назначение профилей групп доступа
Для разграничения доступа и полномочий по функциональности сотрудников, необходимо назначить пользователям соответствующие профили доступа: 

- Профиль доступа "Специалист": Назначается сотрудникам, осуществляющим обслуживание клиентов. Создание документов «Обслуживание клиента», просмотр личных начислений;

- Профиль доступа "Менеджер": Назначается сотрудникам осуществляющим непосредственные работы для клиентов. Создание и редактирование документов «Обслуживание клиента»;

- Профиль доступа "Бухгалтер ИТ-фирмы": Назначается сотрудникам бухгалтерии, работающим с первичной документацией. Работа с документами покупки-продажи, доступ к массовому  формированию закрывающих документов по абонентским договорам;

- Профиль доступа "Кадровик-расчетчик": Назначается кадровику, главному бухгалтеру или сотруднику, занимающемуся расчетом заработной платы и планированием отпусков работников. Полный цикл расчета зарплаты, начисление премий, планирование и анализ отпусков.


------

## 2. Настройка оповещений в Telegram

Настройка оповещений в Telegram

Реализована возможность автоматической отправки уведомлений специалистам о проведении назначенных работ клиентам.

Для настройки необходимо выполнить следующие действия:

1. Создание бота: Через бота @BotFather создается технический бот с возможностью получения уникального API-токен.
2. Создание группы: Создать общую группу в Telegram и добавьте туда бота в качестве администратора.
3. Получение ID: Узнайте идентификатор (ID) созданной группы (например, через бота @myidbot).

Логика работы: Оповещения будут приходить в группу автоматически при создании нового документа «Обслуживание клиента», а также при изменении Даты или Специалиста в уже существующем документе.

------

## 3. Заполнение ключевых констант

Для корректной работы отправки уведомлений Telegram-бота были добавлены следующие «Константы»:

 - Константа "Номенклатура работы специалиста": Используется для автоматического подбора выполненных работ в акты реализации. Устанавливается в параметры Константы из Справочника Номенклатуры(Услуга должна быть внесена в в справочник "Номенклатура" - "Номенклатурная Группа" - "Услуги покупателям").

- Константа "Номенклатура абонентская плата": Используется для ежемесячного фиксированного обслуживания и корректной работы Обработки по выставлению актов. Устанавливается в параметры Константы из Справочника Номенклатуры (услуга должна быть внесена в в справочник "Номенклатура" - "Номенклатурная Группа" - "Абонентское обслуживание").

- Константа "Токен управления ТГБотом": Фиксирует полученный API-токен бота.
- Константа "Идентификатор группы для оповещения": Фиксирует полученный ID группы Telegram.

------

## 4. Настройка графиков и параметров расчета

Перед началом расчетов необходимо заполнить информацию в справочнике "Графики работ" :

В справочнике должен быть создан актуальный график (например, «5-2»).
В регистре сведений «Графики работы» должны быть данные о рабочих часах на текущий год. 
В регистре «Условия оплаты сотрудников» должны быть установлены актуальные размеры окладов и процентов премий по каждому сотруднику.

------

## 5. Особенности работы.

Установлен контроль в документа «Начисление зарплаты» который не допускает создание дублирующих строк для одного сотрудника на один и тот же период по видам расчета «Оклад» и «Отпуск» одновременно. 
В документе «Выплата зарплаты» установлен контроль наличая накопленного долга перед сотрудником.
В документе «График отпусков» реализована проверка лимита в 28 дней и визуализацию на Диаграмме Ганта.

------

## 6. Аналитическая отчетность

Пользователям доступны следующие отчёты в подсистеме «Добавленные объекты», а также в подсистемах «Обслуживание клиентов» и «Зарплаты и Кадры»:

- Анализ выставленных актов (контроль соответствия работ и реализаций).
- Выработка специалистов (анализ эффективности и начисленных премий).
- Расчеты с сотрудниками (сводная ведомость по начислениям и выплатам).
- Расход запланированных отпусков (сравнение плана и факта).

В регистре сведений "Условия оплаты сотрудников" обязательно внесите информацию об окладе (55 000,00) и проценте от работ (20).
